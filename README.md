# Weather Dashboard

A simple weather dashboard built with JavaScript using the OpenWeather API.

## Features
- Search weather by city
- Displays temperature in Celsius
- Fetches real-time weather data using API

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## How to Run
1. Download the project
2. Open `index.html` in your browser
3. Enter a city name to see the weather
